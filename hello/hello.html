HELLO
An R function to say "Hello, world!"

HELLO is an R function which says "Hello, world!".

If you're just trying to learn R, or learning how to use R on a different computer system, or in batch mode, it's helpful if you start with a very simple program with a tiny amount of output that should print immediately if things are working well.
